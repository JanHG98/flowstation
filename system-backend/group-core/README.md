# Group Core

## Zweck

Der Group Core verwaltet alle statischen und dynamischen Gruppeninformationen.

## Kernaufgaben

- GSSI, Gruppennamen und Mitgliedschaften verwalten
- Aktive Gruppenaffiliationen abbilden
- DGNA und Gruppenrufgebiete steuern
- Prioritäten und Berechtigungen bereitstellen
- Beteiligte TBS für Gruppenrufe ermitteln

## Abgrenzung

Keine lokale Timeslot-Zuteilung, keine Sprachverteilung und keine direkte Funk-PDU-Erzeugung.

## WebUI zur Verwaltung

Der Group Core erhält eine eigene Verwaltungsoberfläche für Gruppen, Mitgliedschaften und DGNA.

### Geplante Ansichten

- GSSI, Gruppenname, Alias und Organisation
- statische und dynamische Mitgliedschaften
- aktuell aktive Affiliationen
- Gruppenrufgebiete und Prioritätsprofile
- DGNA-Aufträge und deren Ergebnis
- Import, Export, Audit und Abhängigkeiten

### Kritische Aktionen

- Gruppen anlegen, ändern oder löschen
- Teilnehmer zuweisen oder entfernen
- DGNA auslösen oder zurücknehmen
- Gruppenrufgebiete und Rechte ändern
