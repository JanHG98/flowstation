# Group Core

## Zweck

Der Group Core verwaltet alle statischen und dynamischen Gruppeninformationen.

## Kernaufgaben

- GSSI, Gruppennamen und Mitgliedschaften verwalten
- Aktive Gruppenaffiliationen abbilden
- DGNA und Gruppenrufgebiete steuern
- Prioritäten und Berechtigungen bereitstellen
- Beteiligte TBS für Gruppenrufe ermitteln

## Abgrenzung

Keine lokale Timeslot-Zuteilung, keine Sprachverteilung und keine direkte Funk-PDU-Erzeugung.
