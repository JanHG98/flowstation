# Media Library

## Zweck

Die Media Library verwaltet Audioinhalte, TTS-Dateien und vorbereitete Aussendungen.

## Kernaufgaben

- WAV-/MP3-Bibliothek und TTS-Vorlagen
- NFS- und Archivzugriff
- Audio-Metadaten, Vorverarbeitung und Codec-Cache
- Medien für den Media Switch bereitstellen

## Abgrenzung

Keine aktive Rufsteuerung und kein Recorder-Archiv.

## WebUI zur Verwaltung

Die Media Library erhält eine eigene Verwaltungsoberfläche für Audio, TTS und Vorlagen.

### Geplante Ansichten

- WAV-/MP3-Bibliothek
- TTS-Stimmen und Vorlagen
- Upload, Vorschau und Metadaten
- NFS-/Storage-Zustand
- Codec-Cache und Vorverarbeitung
- Aussendeaufträge an den Media Switch

### Kritische Aktionen

- Dateien hochladen, ersetzen oder löschen
- TTS erzeugen und freigeben
- Vorlage bearbeiten
- Storage neu einlesen oder Cache leeren
